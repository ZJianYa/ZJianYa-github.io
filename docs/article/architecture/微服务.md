# 概念

多个系统集成提供服务  
更多参考 Spring cloud 相关  

## 优点

- 异构
- 弹性
- 扩展  
  按需扩容，包括促销活动  
- 易于部署  
- 与组织结构对其  
  按照业务部分，做研发队伍分割  
- 可组合型  
- 可替代性  

## 缺点

- 分布式事务  
- 开发测试的复杂性  
- 部署、监控的复杂性  
- 故障恢复

## 架构演进
