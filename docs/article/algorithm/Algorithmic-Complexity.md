# 概述

## 度量/复杂度

### 时间复杂度

算法中基本操作次数是问题规模n的某个函数，用T(n)表示。  
而f(n)则表示T(n)的渐进，T(n)=O(f(n))，其中c*f(n)可以表示上界（其中n>2，n需要足够大），所以大O表示上界。  
响应的还Ω和Θ分别表示下界和  
常数复杂度表示为O(1)  
对数复杂度统一表示为O(logN)，忽略底数，因为
多项式复杂度表示为O(n^c)  
线性复杂度表示为O(n)
指数复杂度O(2^n)

O(logN) < O(n) < O(n^c) < O(2^n)
